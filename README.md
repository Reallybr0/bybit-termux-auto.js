# 🚀 Bybit Termux Automation Bot

### 📊 Termux Setup Quick-Reference

| Step | Action | Command |
| :--- | :--- | :--- |
| 1 | Update System | `pkg update -y && pkg upgrade -y` |
| 2 | Install Core Tools | `pkg install nodejs git android-tools -y` |
| 3 | Project Setup | `mkdir bybit-bot && cd bybit-bot && npm init -y` |
| 4 | Install Libraries | `npm install bybit-api readline-sync` |
| 5 | Run Bot | `node bybit_bot.js` |
